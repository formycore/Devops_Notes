Create docker file to perform below steps
    - update centos with latest version
    - install apache httpd webserver and supporting tools
    - Creating custom index image
    - Start apache in the background inside the container
Build the image
Start the container with new Build image
Validate if container is running
Check the container IP
Browse the index page
----------------------------------
2) Modifying the docker image
-  EntryPoint instructs docker to identify which executables should be run when a container is started from our image
    - mkdir docker
    - cd docker
    - vim docker
FROM centos:latest
RUN yum update -y
RUN yum install -y httpd 
RUN echo "Welcome to samantha fans page" > /var/www/html/index.html
ENTRYPOINT apachectl "-DFOREGROUND"
docker build -t centos/apache:v1
-------------
docker images
docker run -d --name apache_web -p 8080:80 centos/apache:v1
docker inspect apache_web | grep -i ipaddress
